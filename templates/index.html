FactionView

competetive network analysis

FactionView is an innovative network analysis tool that uses machine learning techniques to visualize coalition dynamics in complex multi-agent systems. The graph of relationships between agents in a given system is projected onto a two-dimensional plane, producing a kind of strategic map of that system in which agents with more similar relationships appear closer together. This is a generalized visualization technique, and it can be applied to any domain in which there is data that can be modeled as a set of competitive or cooperative relationships.

Originally developed to analyze political factions in Congress, this technique can be applied to the stock market by looking at correlations in stock price changes, revealing the hidden structure of market dynamics. This allows investors and corporate decision-makers to visualize the "big picture" of the economy as a system of interconnected firms, and provides useful insights about the roles that individual firms play within the economy.

Examples:

FactionView Markets — S&P 500 market structure based on correlations of daily stock price changes

FactionView Politics — coalition structures in the U.S. Congress based on voting patterns
